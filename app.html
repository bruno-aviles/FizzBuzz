FIZZBUZZ App

When it counts you Fizz Buzz. A JavaScript coding exercise using loops. A program that prints the numbers from 1 to 100. For multiples of '3' display 'Fizz' and for the multiples of '5' display 'Buzz'. If a multiple of both, display "FizzBuzz".

Instructions

See if you can make FIZZBUZZ appear! Enter a number from 1 to 100 in each input field. Click on submit and see the results below.
